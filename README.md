## Overview

**LevelUP** is a sports-focused e-commerce website where users can buy **accessories, clothing, and nutrition products**, with support for adding **coaches** to the platform.  
As our **first project in this field**, we focused on **dynamic behavior, smooth user experience**, and a **modern, edgy design**, while covering all core e-commerce features.

---

## Technologies Used

The website was built **from scratch** using:

- **PHP**
- **JavaScript**
- **CSS**
- **SQL**

---

## Features

### User Side

- Browse all **active products**
- Deactivated products are hidden from users and visible only to admins
- Dynamic **delivery pricing based on user location**
- Full e-commerce functionality:
  - Shopping cart
  - Wishlist
  - Product reviews
  - Order management

---

### Admin Side

- Admins can also act as **regular clients** using the same account
- Manage:
  - Products
  - Variants
  - Stock
  - Users
- Access to **dashboards and statistics** for analyzing the economic state of the platform
- Advanced control over product visibility and inventory
